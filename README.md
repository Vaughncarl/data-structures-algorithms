# Data Structures & Algorithms

## Description
This repository contains practice implementations of fundamental algorithms and data structures in Python.  
It is designed to demonstrate problem-solving skills and coding fundamentals.

---

## Folders & Files

### sorting
- `bubble_sort.py` — Basic bubble sort implementation
- `quick_sort.py` — Quick sort implementation

### searching
- `binary_search.py` — Binary search on a sorted array

### structures
- `stack.py` — Stack data structure using list
- `queue.py` — Queue data structure using list
- `linked_list.py` — Singly linked list implementation
